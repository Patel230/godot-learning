#WHAT IS GODOT?

Godot is a game engine — a toolkit that helps you build games. Instead of writing code to draw every pixel and play every sound, you get a ready workshop with all the tools.

Free. Open source. Uses GDScript (a language like Python, but built for games). Runs on Windows, Mac, Linux, Web, Mobile.

>> KITCHEN ANALOGY
A game engine is like a professional kitchen. You don't grow wheat or forge pans. You just cook (write code). Godot gives you the stove, tools, recipes — you make the dish.

#SCENES & NODES

>> LEGO ANALOGY
Node = one Lego brick (one job)
Scene = what you build with bricks (a car, a house)
Tree = the blueprint of how bricks connect

Node is the smallest thing. A Timer counts. A Sprite2D shows art. A Label writes text. Everything is a node. Nodes form a tree (parent → children). The root is the boss.

OUR GAME'S TREE

Main (Node) | +-- Player (Area2D) | +-- CollisionShape2D | +-- StartPosition (Marker2D) | +-- MobTimer (Timer) -- spawns a mob every 1.5s +-- ScoreTimer (Timer) -- +1 point every 1s +-- StartTimer (Timer) -- 2s delay before game starts | +-- HUD (CanvasLayer) +-- ScoreLabel +-- MessageLabel +-- MessageTimer +-- [Start] Button
>> OFFICE ANALOGY
Main is the manager. Player is the employee. Timers are alarm clocks. HUD is the receptionist.

main.gd — The Boss

Controls everything: start game, game over, spawn mobs, score.

>> REFEREE
"Game on!" Whistle when hit. Keep score. Send enemies.
>> MAIN.GD
extends Node

@export var mob_scene: PackedScene  # mob.tscn linked here
var score: int

func _ready():
    randomize()
    $Player.hit.connect(game_over)     # Listen for "I'm hit!"
    $HUD.start_game.connect(new_game)    # Listen for "Start!"

func new_game():
    score = 0
    get_tree().call_group("mobs", "queue_free")  # Clear old mobs
    $Player.start($StartPosition.position)        # Reset player
    $StartTimer.start()                           # 2s countdown
    $HUD.update_score(score)
    $HUD.show_message("Get Ready...")

func game_over():
    $MobTimer.stop()         # Stop spawning
    $ScoreTimer.stop()       # Stop scoring
    get_tree().call_group("mobs", "queue_free")
    $HUD.show_game_over()

func _on_mob_timer_timeout():
    var mob = mob_scene.instantiate()  # New enemy spawned
    add_child(mob)

func _on_score_timer_timeout():
    score += 1
    $HUD.update_score(score)

func _on_start_timer_timeout():
    $MobTimer.start()
    $ScoreTimer.start()

player.gd — The Character

Reads WASD/Arrows, moves the green circle, screams when hit.

>> CONTROLLER
Reads your button mashing, moves character, yells "OUCH!"
>> PLAYER.GD
extends Area2D

signal hit

@export var speed = 400
var screen_size: Vector2

func _ready():
    screen_size = get_viewport_rect().size
    area_entered.connect(_on_area_entered)

func _process(delta):
    var velocity = Vector2.ZERO
    if Input.is_key_pressed(KEY_RIGHT) or Input.is_key_pressed(KEY_D):
        velocity.x += 1
    if Input.is_key_pressed(KEY_LEFT) or Input.is_key_pressed(KEY_A):
        velocity.x -= 1
    if Input.is_key_pressed(KEY_DOWN) or Input.is_key_pressed(KEY_S):
        velocity.y += 1
    if Input.is_key_pressed(KEY_UP) or Input.is_key_pressed(KEY_W):
        velocity.y -= 1
    position += velocity * speed * delta
    position = position.clamp(Vector2.ZERO, screen_size)

func start(pos):
    position = pos; show(); $CollisionShape2D.disabled = false

func _on_area_entered(area):
    hide()
    $CollisionShape2D.set_deferred("disabled", true)
    hit.emit()  # "RING RING! I GOT HIT!"

func _draw():
    draw_circle(Vector2.ZERO, 28, Color(0.2, 0.8, 0.2))
    draw_circle(Vector2.ZERO, 18, Color(0.3, 0.9, 0.3))

>> WHY _process(delta) MATTERS

>> HEARTBEAT
_process runs every frame (~60 times/sec). delta is the time between heartbeats. If the game lags, delta gets bigger, so movement stays correct. No delta = your character slows down when the game chokes.

mob.gd — The Enemy

Spawns at random edge, moves in a line, vanishes off-screen.

>> ROOMBA
Appears from nowhere. Moves straight. Disappears forever.
>> MOB.GD
extends Area2D

var speed: float
var direction: Vector2
var screen_size: Vector2

func _ready():
    add_to_group("mobs")  # Tag for mass deletion
    screen_size = get_viewport_rect().size
    var edge = randi() % 4  # 0=top, 1=bottom, 2=left, 3=right
    match edge:
        0:
            position = Vector2(randf_range(0, screen_size.x), -50)
            direction = Vector2.DOWN.rotated(randf_range(-PI/6, PI/6))
        1:
            position = Vector2(randf_range(0, screen_size.x), screen_size.y + 50)
            direction = Vector2.UP.rotated(randf_range(-PI/6, PI/6))
        2:
            position = Vector2(-50, randf_range(0, screen_size.y))
            direction = Vector2.RIGHT.rotated(randf_range(-PI/6, PI/6))
        3:
            position = Vector2(screen_size.x + 50, randf_range(0, screen_size.y))
            direction = Vector2.LEFT.rotated(randf_range(-PI/6, PI/6))
    speed = randf_range(150, 300)
    rotation = direction.angle()

func _process(delta):
    position += direction * speed * delta
    if position.x < -200 or position.x > screen_size.x + 200 \
        or position.y < -200 or position.y > screen_size.y + 200:
        queue_free()  # Delete when gone

func _draw():
    draw_rect(Rect2(-20, -20, 40, 40), Color(0.8, 0.2, 0.2))
    draw_rect(Rect2(-15, -15, 30, 30), Color(0.9, 0.3, 0.3))

hud.gd — The Scoreboard

Score display, messages, Start button.

>> SCOREBOARD
Shows score, "Get Ready", "Game Over".
>> HUD.GD
extends CanvasLayer

signal start_game

func _ready():
    $MessageLabel.text = "Dodge the Creeps"
    $MessageLabel.show()
    $Button.show()

func show_message(text):
    $MessageLabel.text = text
    $MessageLabel.show()
    $MessageTimer.start()  # Auto-hide after 2s

func show_game_over():
    show_message("Game Over")
    await $MessageTimer.timeout  # Wait 2s
    $Button.show()

func update_score(s):
    $ScoreLabel.text = str(s)

func _on_button_pressed():
    $Button.hide(); $MessageLabel.hide()
    start_game.emit()

func _on_message_timer_timeout():
    $MessageLabel.hide()

>> UNDERSTANDING await

>> BUS STOP
await says "wait here for the bus". In show_game_over(), we await the MessageTimer (2 seconds), then show the button. The game doesn't freeze — animations keep playing while we wait.

#KEY CONCEPTS

1. SCENES = RECIPES

A scene is a recipe. Lists ingredients (nodes) and how they connect. mob.tscn is the recipe. Every instantiate() bakes a new cookie.

2. SIGNALS = PHONE CALLS

Node A calls. Node B picks up. hit.emit() = "RING RING!" Main picks up: game_over() = "I'LL HANDLE THIS!" Keeps code separate and clean.

3. COLLISION LAYERS = VIP LISTS

Player: layer=1 ("list 1"), mask=2 ("cares about list 2") Mob: layer=2 ("list 2"), mask=0 ("doesn't care") Mob (list 2) enters Player (cares about list 2): BOOM! Collision! HIT! GAME OVER!

4. _draw() = PAINTING

Instead of image files, you paint with code. Circles, rectangles, lines. queue_redraw() = "paint again!"

_draw() paints layers. First outer, then inner = 2-tone circle.

5. GROUPS = STICKY NOTES

add_to_group("mobs") = sticky note "I'M A MOB!". call_group("mobs", "queue_free") = find all sticky notes → delete all.

6. delta = FRAME TIME

Time since last frame (~1/60 = 0.016s). Multiply speed by delta = movement is smooth at 30 FPS or 144 FPS.

#TRY IT YOURSELF

TWEAKS

  • Faster mobs — In mob.gd change randf_range(150, 300) to randf_range(200, 500)
  • More chaos — In main.tscn change MobTimer wait_time = 1.5 to 0.5
  • You're slower — In player.gd change speed = 400 to 200
  • New colors — Edit _draw() color values. Make a pink player. Blue mobs.
  • Shapeshifter — Replace draw_circle with draw_rect or vice versa

#GLOSSARY

NODEOne building block. Timer, Sprite, Label — everything is a node.
SCENEA file containing a tree of nodes. Like a saved recipe.
SIGNALEvent notification. One node calls, another handles. Like a phone.
GDSCRIPTGodot's language. Like Python with game superpowers.
_process()Runs every frame (~60Hz). The game's heartbeat.
deltaSeconds since last frame. Keeps movement frame-rate independent.
Area2D2D zone that detects overlaps and collisions.
CollisionShape2DThe actual hitbox shape: circle, rectangle, polygon.
CanvasLayerUI layer on top of the game world. HUD lives here.
queue_free()Safely deletes a node at end of frame.
instantiate()Copies a scene into the game. Like a cookie cutter.
@exportMakes a variable editable in the Inspector panel.
